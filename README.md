# 🛡️ Bitdefender Antivirus Website

A modern, responsive website for Bitdefender Antivirus built with React, TypeScript, and Tailwind CSS. Features multiple pages, authentication system, and live chat integration.

## 🚀 Live Demo

[View Live Demo](https://your-domain.com)

## ✨ Features

- **Modern UI/UX** - Beautiful, responsive design with smooth animations
- **Multi-page Layout** - Home, Features, Pricing, Plans, FAQ, About, Contact
- **Authentication System** - Login, Signup, and Password Reset functionality
- **Live Chat Integration** - 24/7 customer support link
- **Responsive Design** - Perfect on mobile, tablet, and desktop
- **TypeScript** - Type-safe code
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations

## 📋 Pages

| Page | Description |
|------|-------------|
| Home | Hero section, features preview, testimonials |
| Features | All security features with descriptions |
| Pricing | Plan comparison with pricing cards |
| Plans | Detailed plan comparison table |
| FAQ | Frequently asked questions with search |
| About | Company information and team |
| Contact | Contact form and information |
| Login | User authentication |
| Signup | New user registration |

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Routing**: React Router v6
- **Forms**: React Hook Form
- **Icons**: React Icons
- **Notifications**: React Hot Toast
- **Build Tool**: Create React App
